结论：

有基本数据类型base1，然后用base2 = base1的方式给base2赋值，base2改变时不影响base1的值

有引用数据类型obj1 ，然后用obj2 = obj1的方式给obj2赋值，obj2改变时影响obj1的值

注：基本数据类型有 number boolean string ；引用数据类型有 array object
